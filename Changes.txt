Changes for 0.3.3 (2021-1-25)
------------------------------------------------------------------------------
New features:
* run -a コマンドで、自動タギング機能追加。実行結果の類似パターンから比較対象
  サーバを選出してタグ付けする

Changes for 0.3.2 (2021-1-18)
------------------------------------------------------------------------------
Bug fixes:
* 検査結果 Excel を開いた時に、削除されたパーツの XML エラー(致命的)が発生する
  問題対処。
* Redmine 4.0 から 4.1 に上げると、チケット登録で Java Redmine ライブラリの
  互換性エラーが発生する問題の暫定対処。例外メッセージを出力
* run --slient コマンド実行で、ヘッダメッセージが出力される問題対処
* run --test {プラットフォーム}コマンド実行で、絞り込みができない問題対処
