Changes for 0.3.4 (2021-2-1)
------------------------------------------------------------------------------
New features:
* Windows 版エージェントのインベントリ収集用パッケージのビルドスクリプト追加。
  gradle zipWindowsAgentApp でパッケージをビルド

Bug fixes:
* Redmine 接続用パラメータで設定ファイルよりも環境変数が優先される問題対処
  旧バージョンとの共有環境での利用を想定し、設定ファイルを優先させる様に変更

Changes for 0.3.3 (2021-1-25)
------------------------------------------------------------------------------
New features:
* run -a コマンドで、自動タギング機能追加。実行結果の類似パターンから比較対象
  サーバを選出してタグ付けする
* バッチ実行スクリプト getbat に、--test {プラットフォーム}, --dry, 
  --printonly オプション追加

Changes for 0.3.2 (2021-1-18)
------------------------------------------------------------------------------
Bug fixes:
* Excel で検査結果を開いた時に、削除したコメントパーツの XML 
  解析エラーが発生する問題対処。
* Redmine 4.0 から 4.1 に移行すると、チケット登録で redmine-java-api 
  ライブラリの互換性エラーが発生する問題の暫定対処。例外を追加
* getcf run --silent コマンドで、余計なメッセージが出力される問題対処
* getcf run --test {プラットフォーム} コマンドで、プラットフォームの
  絞り込みができない問題対処
